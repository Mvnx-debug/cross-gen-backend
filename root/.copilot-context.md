# Projeto: CrossGen Backend

## Stack
- Bun
- Elysia
- TypeScript (strict)
- Drizzle ORM (PostgreSQL)
- Better Auth
- Docker

## Arquitetura
- Guardas de autenticação e autorização como plugins do Elysia
- `shared/auth` → auth, roles, guards
- `modules/*` → rotas e serviços por domínio
- `shared/db` → Drizzle + schema tipado

## Regras de Código
- Nada de `any`
- Tipagem explícita
- Guards usam `decorate` + `derive`
- Services não conhecem HTTP
- Rotas são protegidas via `.use()`

## Padrão de Git
- Uma feature por branch
- Commits no padrão Conventional Commits
- Avisar quando uma feature estiver pronta para commit
